📤 Figma Slides 批量导出指南

⚠️ 重要提示：
由于 Figma Slides 文件不支持 REST API 访问，需要通过 Figma Desktop App 手动导出。
本指南将帮助您快速完成批量导出。

方法 1: 使用 Figma Desktop App（推荐）

步骤 1：打开文件
点击下方按钮在 Figma 中打开文件：
🔗 打开 Figma Slides 📁 打开输出文件夹
步骤 2：选择要导出的幻灯片
在 Figma 左侧面板中：
• 按住 Cmd (Mac) 或 Ctrl (Windows)
• 依次点击要导出的幻灯片/页面
• 可以一次性选择所有幻灯片
步骤 3：批量导出
在右侧属性面板中：
1. 找到 Export 部分
2. 点击 + 添加导出设置
3. 选择格式：PNG
4. 选择缩放：2x (高清)
5. 点击 Export [N] layers
6. 选择保存位置：/Users/aa123/Documents/presentation skills/MAI Design Skill/asset/Images
✅ 导出完成后：
所有图片将保存在 Images 文件夹中
文件名将使用 Figma 中的图层名称

方法 2: 使用快捷键快速导出

快捷键导出流程：
1. 选择幻灯片/元素
2. 按 Cmd/Ctrl + Shift + E 快速导出
3. 选择保存位置和格式
4. 重复以上步骤导出其他内容

推荐的导出内容

📄 幻灯片页面：
• Cover (封面页)
• 所有内容页面
• 模板布局示例

🎨 视觉元素：
• Logo 和品牌图标
• 背景图案和纹理
• 装饰性插图

🧩 UI 组件：
• Badge 组件
• Label 组件
• Card 组件
• Icon 图标库

文件命名建议

导出后，建议按以下规则重命名：

slide_01_cover.png - 封面
slide_02_hero.png - Hero页面
slide_03_content.png - 内容页
component_badge_yellow.png - Badge组件
component_label_green.png - Label组件
icon_copilot_100px.png - 图标
bg_warm_beige.png - 背景
💡 提示：
如果一次性导出所有幻灯片，Figma 会自动使用图层名称作为文件名。
您可以在导出后批量重命名，或者在 Figma 中先重命名图层。

🔧 技术限制说明

为什么不能用 API 自动导出？
Figma Slides 是一种特殊的文件类型，目前不支持通过 REST API 的 /v1/files/{key} 端点访问。 尝试访问会返回：{"status": 400, "err": "File type not supported by this endpoint"}

已验证的方案：
✅ Figma Desktop App 手动导出
✅ Figma 网页版手动导出
❌ REST API 自动导出（不支持）
❌ Figma Plugin API（Slides 文件限制）
