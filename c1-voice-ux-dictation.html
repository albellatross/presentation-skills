What Users Want to Achieve

Quick Capture

Rapidly turn thoughts into text without breaking concentration. Voice becomes an extension of the keyboard for moments when typing isn't convenient.

Mental Flow

Maintain train of thought while composing. The interface should be invisible — no friction between thinking and capturing.

Usable Output

C1 Voice UX UI Design: Controls

Voice Settings

Show Mico Toggle: Users can enable or disable the voice icon from the settings panel, giving them control over when voice features are accessible.

Language Selection: Auto-detect by default, with option to set a specific language for more accurate transcription.

Voice Selection: Choose from multiple voice options (Birch, Meadow, Canyon, Grove) for audio feedback and responses.

These settings are accessible via Settings → Voice Settings, maintaining the existing navigation pattern.
