Evidence

C1 Voice UX User Problem

"Users misunderstand the 'Microphone' icon. They expect it to start dictation, not a conversation."

C1 Voice UX User Research Quote

Core Insight

Users need hands-free content composition — a dedicated way to turn voice into text.

C1 Voice UX

02

Solution

C1 Voice UX

Dedicated Dictation Entry Point

In the compose experience

A clear, discoverable way to start speech-to-text that users can easily understand and access within any text composition flow.

C1 Voice UX Solution Overview

User Goals

Quick Capture

Rapidly turn thoughts into text without breaking concentration. Voice becomes an extension of the keyboard for moments when typing isn't convenient.

Mental Flow

Maintain train of thought while composing. The interface should be invisible — no friction between thinking and capturing.

Usable Output

Get written content that's immediately useful — properly formatted, punctuated, and ready to send or edit minimally.
